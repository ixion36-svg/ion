7Governance & Storage

Data Stewardship

Departments retain ownership of their data. The SOC acts as custodian. Responsibility for data accuracy, classification, and lawful basis remains with the Department.

Retention Cap

SOC provides hot storage per the retention selected in Section 4. Compliance archiving beyond this period must be managed by the Department. Cold storage is read-only and may have slower query times.

SLA Breach Process

If data drops below the agreed uptime target or field quality degrades, the SOC will raise a ticket. If unresolved within 2 review cycles, escalation to CISO.

Review Cadence

This agreement is reviewed per the cadence set in Section 1. Both parties meet to review data health, field mapping score, use case coverage, and gap remediation progress.
